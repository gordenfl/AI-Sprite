# LangChain

就可以理解是一个语言的链条, 他能是LLM应用程序开发框架.主要的功能:

1. 连接外部数据源 SQL, Notion, PDF 还有向量数据库 (TODO) 等等
2. 调用工具: 比如Google 搜索, 计算器, Python脚本, API调用等等
3. 实现链式逻辑: 结合多个Prompt 的组合在已有的数据中做推断
4. 上下文记忆逻辑: 为Agent或者聊天系统提供对话历史记忆
5. Agent系统: 让语言模型在执行任务是判断下一步用什么工具,或者采取什么操作

    | 组件         | 功能说明                                |
    | ---------- | ----------------------------------- |
    | **LLM**    | 集成 OpenAI、Anthropic、HuggingFace 等模型 |
    | **Prompt** | 提供 Prompt 模板与填充机制                   |
    | **Chains** | 将多个调用按逻辑顺序组织起来（例如提问 → 查询文档 → 回答）    |
    | **Agents** | 具备决策能力的系统，可调用工具完成复杂任务               |
    | **Memory** | 支持保存和调用对话或任务过程的上下文记忆                |
    | **Tools**  | 可注册外部功能供 Agent 使用，如搜索、数据库、代码执行等     |
