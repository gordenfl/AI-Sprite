# AI Commenter

This is a VSCode extension that uses OpenAI API to generate code comments automatically.

## Features

- Select code and use command "AI 添加代码注释" to generate comments above the code.

## Requirements

- Set environment variable `OPENAI_API_KEY` with your OpenAI API key.

## Usage

1. Select a code block;
2. Open command palette and run `AI 添加代码注释`;
3. Wait for the comment to be inserted above your selection.
